EXPENSIFY

Is a web application that allows to keep an expenses control according to some tags like category, type and date. This application also gives you a dashboard where you can see a relation between your expenses in the time, it does all the expenses actions asynchronously via Ajax and additionaly it has a RESTful API where all the CRUD functionalities are implemented. This aplication was developed un Ruby on Rails using Postgress for the database.

BACHILLERATO DIGITAL

This is a digital online high-school platform held out by the government of Antioquia. The platform was planned as a social project in which the people with few academical and educational resources could study their high-school. I was a member in the front end team where we brought all the courses asynchronously using HTML, CSS and Javascript with JQuery.

C4 SECURITY PLATFORM

This is a project developed for the GAULA (Unified Action Groups for Personal Freedom), it is a digital platform where people are taught how to act in different situations like telephone extortions, how to behave with pets, how to act with elderly people, among others. This project was developed in Wordpress and i was in charge of the front end and the handling of the framework's plug-ins with other 3 developers.

BLOG-POST

This is a blog-post web aplication in which it is implemented the authentication, validations and markdown functionalities. I developed this project with ruby on rails using Divise for the authentication and Postgress for the database.

BULLS AND COWS GAME

This game consists in guessing a 4 digit number on several tries, every digit of the number must be different and the program is constantly giving you feedback of how many digits are right but in wrong position and how many are right and in the correct position.
